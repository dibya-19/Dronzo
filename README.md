# Dronzo

## Overview
Dronzo is a cutting-edge drone delivery and tracking system designed to streamline package delivery with real-time tracking and reliable performance. Our platform aims to enhance logistics efficiency by leveraging autonomous drone technology and advanced tracking capabilities.

## Features
- Autonomous drone delivery
- Real-time package tracking
- Secure and reliable delivery system
- User-friendly interface for order management
- Route optimization for efficient deliveries

## Tech Stack
- Backend: Node.js, Express
- Frontend: React.js
- Database: MongoDB
- Mapping & Tracking: Google Maps API
- Deployment: Docker, AWS

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/dronzo.git
   cd dronzo
   ```
2. Install backend dependencies:
   ```bash
   cd backend
   npm install
   ```
3. Install frontend dependencies:
   ```bash
   cd ../frontend
   npm install
   ```
4. Set up environment variables:
   Create a `.env` file in both backend and frontend directories with appropriate configuration.

5. Start the backend server:
   ```bash
   cd ../backend
   npm start
   ```
6. Start the frontend application:
   ```bash
   cd ../frontend
   npm start
   ```

## Usage
- Register or log in to the platform.
- Place a delivery order specifying pickup and drop-off locations.
- Track your package in real-time on the dashboard.
- Receive notifications upon delivery completion.
